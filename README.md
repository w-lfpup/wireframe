# Wireframe

`Wireframe-rs` is a component set built for the web.

## About

Wireframe is mostly a CSS library.

It also has some web components.

## Config

```JSON
{
	"components": [],
	"styles": "./yo/",
}
```

### Foundations

Interactivity by color.

lightest gray - interactable
gray - highlighted
dark gray - pressed
black - selected / checked

## Components

### Inputs

- checkbox
- radio
- switch
- date
- time
- text
- email
- phone

### Actions

- chips
- buttons
- action buttons
- segmented buttons
- menus

### Containment

- cards
- dialogs
- divider
- lists
